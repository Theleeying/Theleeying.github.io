## 🎵 项目简介

EchoVerse 是一个基于纯前端技术构建的现代化音乐网站，集成了GD音乐台API，提供音乐搜索、在线播放、歌词同步、歌单管理等功能。网站采用响应式设计，完美支持桌面端、平板和移动端访问。

### 🌟 项目亮点
- 🎧 **高品质音频播放** - 支持320kbps高品质音频
- 🎤 **实时歌词同步** - 支持LRC格式歌词显示
- 📱 **完全响应式** - 适配所有设备尺寸
- 💾 **智能状态保存** - 切换页面不中断播放
- 🎯 **智能推荐系统** - 基于音乐风格的个性化推荐
- ⌨️ **丰富快捷键** - 支持键盘快捷操作

## 🔌 API集成详情

### GD音乐台API集成
本项目深度集成了GD音乐台API，提供完整的音乐服务体验。

#### API接口说明
| 接口类型 | 请求地址 | 参数 | 功能 |
|---------|---------|------|------|
| 音乐搜索 | `/api.php?types=search` | `name`, `count`, `pages` | 搜索歌曲、歌手、专辑 |
| 播放链接 | `/api.php?types=url` | `id`, `br` | 获取音频播放链接 |
| 专辑封面 | `/api.php?types=pic` | `id`, `size` | 获取专辑封面图片 |
| 歌词数据 | `/api.php?types=lyric` | `id` | 获取LRC格式歌词 |

#### 技术实现
- **错误处理**: 完善的API错误处理和重试机制
- **数据缓存**: 智能缓存策略，减少API调用
- **响应解析**: 支持多种API响应格式的自动解析
- **网络优化**: 请求超时设置和网络状态检测
- **兼容性**: 支持CORS跨域请求

#### 数据流程
1. **搜索流程**: 用户输入 → API搜索 → 结果解析 → 界面展示
2. **播放流程**: 选择歌曲 → 获取播放链接 → 验证URL → 开始播放
3. **封面流程**: 歌曲信息 → 获取封面ID → 请求封面 → 预加载显示
4. **歌词流程**: 播放歌曲 → 获取歌词ID → 请求歌词 → 解析同步

### 第三方服务
- **GD音乐台API** - 音乐数据源，提供搜索、播放、歌词等服务
  - **API地址**: https://music-api.gdstudio.xyz/
  - **官方网站**: https://music.gdstudio.xyz/
  - **音乐搜索接口** - 支持歌曲、歌手、专辑搜索
  - **音频播放链接获取** - 支持320kbps高品质音频
  - **专辑封面获取** - 自动获取专辑封面图片
  - **歌词数据获取** - 支持LRC格式歌词同步
  - **数据来源**: 网易云音乐、QQ音乐等主流平台

### 开发工具
- **模块化架构** - 按功能分离的JavaScript模块
- **响应式设计** - 移动端优先的响应式布局

```javascript

  id: 'user_id',

  email: 'email@example.com'
}));

// 播放历史
localStorage.setItem('playHistory_user_id', JSON.stringify([
  {
    id: 'song_id',
    title: 'song_title',

    cover: 'cover_url',
    url: 'audio_url',
    playTime: '2025-01-01T12:00:00Z',

// 播放器状态
localStorage.setItem('musicPlayerState', JSON.stringify({
  currentSong: {...},

  isMuted: false,
  isPlaying: false,
  playlist: [...],

}));
```
