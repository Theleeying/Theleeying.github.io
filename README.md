## 🎵 项目简介

EchoVerse 是一个基于纯前端技术构建的现代化音乐网站，集成了GD音乐台API，提供音乐搜索、在线播放、歌词同步、歌单管理等功能。网站采用响应式设计，完美支持桌面端、平板和移动端访问。

### 🌟 项目亮点
- 🎧 **高品质音频播放** - 支持320kbps高品质音频
- 🎤 **实时歌词同步** - 支持LRC格式歌词显示
- 📱 **完全响应式** - 适配所有设备尺寸
- 💾 **智能状态保存** - 切换页面不中断播放
- 🎯 **智能推荐系统** - 基于音乐风格的个性化推荐
- ⌨️ **丰富快捷键** - 支持键盘快捷操作

## ✨ 主要功能

### 🎯 核心功能
- **🔍 智能音乐搜索** - 支持歌曲名、歌手名、专辑名搜索，集成GD音乐台API
- **🎧 高品质在线播放** - 支持320kbps高品质音频播放
- **🎤 实时歌词同步** - 支持LRC格式歌词，点击专辑封面查看歌词
- **📋 动态歌单管理** - 基于音乐风格的智能歌单推荐
- **🎨 专辑展示** - 展示热门艺术家专辑，点击获取歌手歌曲
- **💾 智能状态保存** - 使用LocalStorage保存播放状态和历史

### 🏠 首页功能
- **🎵 推荐歌单** - 动态加载不同风格歌单（流行、摇滚、民谣、电子等）
- **🎨 新专速递** - 展示热门艺术家专辑（周杰伦、王力宏、邓紫棋等）
- **🔄 分类筛选** - 支持按音乐风格筛选歌单
- **📱 响应式布局** - 完美适配各种设备

### 🔍 发现页面
- **🔎 实时搜索** - 输入关键词即时搜索音乐
- **🎵 搜索结果** - 展示歌曲信息、专辑封面、播放时长
- **▶️ 一键播放** - 点击歌曲直接播放
- **🖼️ 封面预加载** - 防止图片闪烁，提升用户体验

### 🎵 播放器功能
- **▶️ 播放控制** - 播放/暂停、上一首/下一首
- **📊 进度控制** - 拖拽进度条、时间显示
- **🔊 音量控制** - 音量调节、智能静音
- **🎤 歌词显示** - 点击专辑封面查看同步歌词
- **⌨️ 快捷键支持** - 空格键播放/暂停，方向键控制进度和音量
- **💾 状态持久化** - 切换页面保持播放状态

## 🚀 技术栈

### 前端技术
- **HTML5** - 语义化页面结构，使用现代HTML5标签
- **CSS3** - 现代样式设计，包含渐变、动画、响应式布局
- **JavaScript ES6+** - 模块化开发，使用类、箭头函数、async/await
- **Fetch API** - 异步网络请求，处理API数据
- **Web Audio API** - 音频播放控制和事件处理
- **LocalStorage** - 客户端数据持久化存储

### 第三方服务
- **GD音乐台API** - 音乐数据源，提供搜索、播放、歌词等服务
  - 音乐搜索接口
  - 音频播放链接获取
  - 专辑封面获取
  - 歌词数据获取

### 开发工具
- **模块化架构** - 按功能分离的JavaScript模块
- **响应式设计** - 移动端优先的响应式布局
- **性能优化** - 图片预加载、状态缓存、事件节流

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome 60+, Firefox 55+, Safari 12+, Edge 79+）
- 本地Web服务器（推荐使用Live Server或类似工具）

### 安装步骤
1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/EchoVerse.git
   cd EchoVerse
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server
   
   # 或使用Live Server扩展（推荐）
   ```

3. **访问网站**
   ```
   http://localhost:8000
   ```

### 使用说明
1. **首页** - 浏览推荐歌单和热门专辑
2. **发现** - 搜索你喜欢的音乐
3. **播放** - 点击歌曲开始播放，支持歌词显示
4. **快捷键** - 使用键盘快捷键控制播放
